# AAE5302 Assignment 2 – ORB-SLAM3 Trajectory Evaluation

## Overview

This repository contains the results for **Assignment 2 of AAE5302**.  
The objective of this assignment is to run **ORB-SLAM3** on the **HKIsland dataset**, generate the estimated camera trajectory, and evaluate it against the ground truth trajectory.

The evaluation compares the estimated trajectory produced by ORB-SLAM3 with the provided ground truth trajectory using trajectory evaluation tools.

---

## Repository Structure

The repository contains the following files:

### Estimated Camera Trajectory

[CameraTrajectory.txt](CameraTrajectory.txt)

This file contains the **estimated camera trajectory** generated by ORB-SLAM3 during the SLAM process.

---

### Ground Truth Trajectory

[ground_truth_HKisland.txt](ground_truth_HKisland.txt)

This file provides the **ground truth trajectory** for the HKIsland dataset.  
It is used as the reference trajectory to evaluate the accuracy of the SLAM result.

---

### Evaluation Results

[FINAL_EVALUATION_RESULTS.txt](FINAL_EVALUATION_RESULTS.txt)

This file contains the **numerical evaluation results**, including trajectory error metrics calculated by the evaluation tool.
